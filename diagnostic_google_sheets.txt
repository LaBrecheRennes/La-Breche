DIAGNOSTIC DU CHARGEMENT DES CERCLES DEPUIS GOOGLE SHEETS
===============================================

Informations générales :
- ID de la feuille Google : 1klaUaZOED9tztQkSrLxlG-bOSbChAx95jZbCjqNEBaY
- URL d'accès : https://docs.google.com/spreadsheets/d/1klaUaZOED9tztQkSrLxlG-bOSbChAx95jZbCjqNEBaY

Vérifications à effectuer :
1. La clé API Google Sheets doit être valide et avoir accès à la feuille
2. La feuille doit comporter au moins un onglet différent de "Historique des cercles"
3. Les onglets doivent suivre la structure suivante :
   - Colonne B : clés (ex: "date", "lieu", "capacité_totale", etc.)
   - Colonne C : valeurs correspondantes
   - Colonne E : liste des participants (le système compte -2 pour le nombre réel)

Pour voir les logs détaillés :
1. Ouvrez la console du navigateur (F12 puis onglet "Console")
2. Rechargez la page et observez les messages

Messages d'erreur courants :
- 401 : Problème d'authentification avec la clé API
- 403 : La clé API n'a pas les permissions suffisantes
- 404 : La feuille n'existe pas ou n'est pas accessible

Comment corriger les erreurs :
1. Vérifiez que la clé API est correcte
2. Assurez-vous que la clé API a accès à l'API Google Sheets
3. Vérifiez que l'ID de la feuille est correct
4. Assurez-vous que la feuille contient des onglets avec la structure requise

Rapport de diagnostic automatique :
Exécutez la page et vérifiez la console pour voir les informations de diagnostic complètes, 
incluant le nombre total d'onglets, les noms des onglets filtrés, et le nombre de cercles chargés.
